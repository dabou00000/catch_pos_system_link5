🎯 الحل الأول: زر تحديث يدوي

1. استخدم الزر الجديد

تم إضافة زر "تحديث الأسعار" في نقطة البيع بجانب زر "مسح العربة"

💡 بعد تعديل سعر منتج، انقر على زر "تحديث الأسعار" في نقطة البيع

🔧 الحل الثاني: تحديث يدوي من Console

1. افتح Developer Tools

اضغط F12 أو انقر بزر الماوس الأيمن → "فحص العنصر"

2. اذهب إلى Console

انقر على تبويب "Console" في Developer Tools

3. انسخ والصق هذا الكود

forceRefreshPOS();

ثم اضغط Enter

✅ هذا سيحدث جميع الأسعار في نقطة البيع فوراً!

🔍 الحل الثالث: تشخيص المشكلة

إذا لم تعمل الحلول أعلاه:

انسخ والصق هذا الكود في Console للتشخيص:

// تشخيص شامل للمشكلة console.log('🔍 بدء التشخيص...'); // 1. فحص localStorage let storedProducts = JSON.parse(localStorage.getItem('products') || '[]'); console.log('📦 المنتجات في localStorage:', storedProducts.length); // 2. فحص متغير products console.log('📦 المنتجات في المتغير:', products ? products.length : 'غير موجود'); // 3. فحص العربة console.log('🛒 المنتجات في العربة:', cart ? cart.length : 'غير موجود'); // 4. فحص منتج معين let testProduct = storedProducts.find(p => p.name.includes('t-shirt')); if (testProduct) { console.log('🧪 منتج الاختبار:', testProduct.name); console.log(' السعر الحالي:', testProduct.priceUSD); console.log(' الأسعار المتعددة:', testProduct.prices); } // 5. فحص المنتج في العربة if (cart && cart.length > 0) { let cartProduct = cart.find(item => item.name.includes('t-shirt')); if (cartProduct) { console.log('🛒 المنتج في العربة:', cartProduct.name); console.log(' السعر في العربة:', cartProduct.priceUSD); } } console.log('✅ انتهى التشخيص');

⚡ الحل الرابع: تحديث مباشر للسعر

إذا كنت تريد تحديث سعر منتج معين مباشرة:

// تحديث سعر منتج معين مباشرة function updateProductPrice(productName, newPrice) { // تحديث في localStorage let products = JSON.parse(localStorage.getItem('products') || '[]'); let product = products.find(p => p.name.includes(productName)); if (product) { product.priceUSD = newPrice; product.priceLBP = Math.round(newPrice * 90000); // افتراض سعر صرف 90000 localStorage.setItem('products', JSON.stringify(products)); console.log(`✅ تم تحديث سعر ${product.name} إلى $${newPrice}`); } // تحديث في العربة if (cart && cart.length > 0) { let cartItem = cart.find(item => item.name.includes(productName)); if (cartItem) { cartItem.priceUSD = newPrice; cartItem.priceLBP = Math.round(newPrice * 90000); console.log(`✅ تم تحديث سعر المنتج في العربة`); } } // إعادة عرض العربة if (typeof updateCart === 'function') { updateCart(); } } // مثال: تحديث سعر t-shirt إلى $35 // updateProductPrice('t-shirt', 35);

⚠️ تحذير مهم

هذه حلول مؤقتة. بعد استخدامها، يجب أن نجد السبب الجذري للمشكلة ونصلحه في الكود الأساسي.

📞 تقرير المشكلة

بعد تجربة الحلول أعلاه، أرسل لي:

  • أي حل عمل معك؟
  • ما هي رسائل Console التي ظهرت؟
  • هل توجد أخطاء؟
  • هل المشكلة لا تزال موجودة؟
